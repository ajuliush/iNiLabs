## Installation

-   `https://github.com/itminanbiva078/hrms_software.git`
-   `cd iNiLabs/`
-   `composer install`
-   `cp .env.example .env`
-   `update .env` and set your database credentials
-   `php artisan key:generate`
-   `npm install`
-   `npm run dev`
-   `php artisan migrate:fresh --seed`
-   `php artisan serve`
